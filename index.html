HTML

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • DRY: Do Not Repeat. Keep code clean and less redundant.
  • ,: a comma can be used to give multiple selections a declaration.
  • CSS Syntax: Selector {Property:Value; Property:Value;} "Property:Value" would be a declaration
  • *: a special selector that applies declaration to everything
  • Separation of Concerns: Separate code into different files based on use.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes or mods to the staging area
  • git commit -m "comment here": adds changes or mods from the staging area to be commited with the included note
  • git pull origin main: updates your local main file with what is in the current git repo version
  • git push origin branch-name: sends commits to git repo branch to be approved for merge
  • git checkout branch-name: switches to branch-name
